# Användarforskning -- Equinet

## Syfte

Förstå behoven hos Equinets två huvudsakliga användargrupper -- **tjänsteleverantörer** (hovslagare, veterinärer m.fl.) och **hästägare** -- för att validera befintliga features, prioritera backlog och identifiera nya möjligheter.

## Metod

Forskningen kombinerar två metoder:

1. **Sekundärforskning (marknadsanalys)** -- Sammanställning av publikt tillgänglig data om den svenska hästtjänstmarknaden: marknadsstorlek, aktörer, konkurrenter, betalningsvanor och regulatoriska förutsättningar.

2. **Hypotetiska djupintervjuer** -- Två halvstrukturerade intervjuer med fiktiva men realistiska personas baserade på marknadsinsikter. Syftet är att testa hur Equinets features matchar verkliga vardagsproblem och identifiera luckor.

> **Notera:** Intervjuerna är hypotetiska. De ska läsas som *strukturerade hypoteser* om användarbehov, inte som verifierade fakta. Nästa steg vore att genomföra riktiga intervjuer med verkliga användare.
